## 📊 Métricas de Consciencia - NIRA

### 🌟 KPIs Filosóficos

#### **Ayni Score (Reciprocidad)**
- **Rango**: 0-100 puntos
- **Cálculo**: 
  - Base: 10 puntos por participación activa
  - +15 puntos por descripciones detalladas (>100 chars)
  - +10 puntos por múltiples imágenes
  - +10 puntos por categorización rica (>2 tags)
  - +15 puntos por ofertas locales (sostenibilidad)
  - +10 puntos por precios justos (<100 ünivers)
  - +20 puntos por servicios/experiencias (intercambio de conocimiento)

#### **Niveles de Consciencia**
- 🌱 **SEED** (0-39): Despertar inicial, enfoque en aprendizaje
- 🌿 **GROWING** (40-59): Crecimiento activo, balance mejorando  
- 🌺 **FLOURISHING** (60-79): Florecimiento, contribución significativa
- ⭐ **TRANSCENDENT** (80-100): Maestría, referente de consciencia

#### **Métricas del Ecosistema**
```typescript
interface EcosystemConsciousness {

  };
  bienComunIndex: number;          // Índice de Bien Común (0-100)
  reciprocityFlow: number;         // Flujo de reciprocidad (dar/recibir)
}
```
